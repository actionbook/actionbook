# Browser Agent

Browser automation agent using [Stagehand](https://github.com/browserbasehq/stagehand) + AI SDK + Actionbook.

## Demo

https://github.com/user-attachments/assets/6c8cea1c-6c78-41e1-9f65-a98eaa0efcea

## Setup

```bash
pnpm install
cp .env.example .env
# Edit .env with your API keys
```

## Run

```bash
pnpm dev
```

Set `HEADLESS=true` to run without browser UI.
